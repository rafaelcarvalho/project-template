## 💬 Getting Help

### GitHub Discussions

For questions, ideas, and community support:

👉 **[Start a Discussion](https://github.com/your-org/your-repo/discussions)**

Use discussions for:
- Architecture questions
- Design pattern clarifications
- Feature requests
- General questions about the codebase
- Sharing how you're using this template

### GitHub Issues

For bug reports and concrete problems:

👉 **[Open an Issue](https://github.com/your-org/your-repo/issues/new)**

Before opening an issue:
1. Search existing issues to avoid duplicates
2. Verify the issue with the latest version
3. Provide a minimal reproduction case

**Good issue template:**

```
## Description
Brief description of the problem

## Steps to Reproduce
1. Clone the repository
2. Run `./gradlew build`
3. Observe error: ...

## Expected Behavior
What should happen

## Actual Behavior
What actually happens

## Environment
- OS: macOS 14.2
- Java: 25.0.2
- Gradle: 9.1.0
```

## 🐛 Reporting Bugs

When reporting bugs, include:

- **Environment details** — OS, Java version, Gradle version
- **Stack traces** — Full error output
- **Logs** — Relevant application logs
- **Reproduction** — Steps to reproduce or a failing test case

Run diagnostics:

```bash
./gradlew --version
java -version
./gradlew dependencies
```

## 💡 Feature Requests

We welcome feature requests! Please:

1. Check if it aligns with our [architectural principles](README.md#-architectural-vision)
2. Open a discussion first to gather feedback
3. Provide clear use cases and motivation
4. Explain how it fits the modular monolith philosophy

## ⏱️ Response Times

We aim for:

- **Security issues** — Acknowledged within 48 hours
- **Critical bugs** — Investigated within 1 week
- **General questions** — Responded to within 2 weeks
- **Feature requests** — Triaged within 1 month

> Note: This is an open-source project maintained by volunteers. Response times are best-effort.

We cannot help with:

- Custom deployments outside the documented setup
- Integration with proprietary systems
- Consulting or architecture services
- Third-party dependencies issues
